Beginner's Tutorial

In this tutorial, you will learn how to create a one page website dedicated to you. It will use basic HTML5 and CSS3 concepts and build upon your understanding of the interaction between HTML and CSS.

Pre-Requisite

In order to follow and do some of the exercises within this tutorial, you will need a text editor. It can be a basic note editor such as as Notepad or TextEdit. Click here to view the Text Editors that we recommend.

Setting Up your HTML Page

The first thing that needs to be done is create your html page. Create a file with the name of "index" and the extension of ".html". Index is used as the filename as most web servers look for index as the homepage. Additionally create a file call "style.css" which will be used in a later section to style the page. Once the file is created, copy and paste the HTML framework code is shown below.

<!DOCTYPE html>
<html>
<head>
<title> </title>
<link href="style.css" rel="stylesheet"> </head>
<body>
</body>
</html>

Once this code has been pasted into the your "index.html" file, you now have a basic, blank webpage.

Setting the Page's Title

Between the Title Tags in your HTML document, enter in your name as we are creating a profile page for yourself. Once entered, your code should look similar to below. Changes are shown in red.

<!DOCTYPE html>
<html>
<head>
<title> Andrew </title>
<link href="style.css" rel="stylesheet"> </head>
<body>
</body>
</html>

If you view your webpage in a web browser, which is recommended, you can see now that the title bar should display your name.

Insert Page Content

In the profile page we will have four main elements: your name in big letters, a picture of you, a short description or biography about yourself and a link to your social media account (i.e. Facebook, Twitter, LinkedIn, etc). First thing is to add your name in big letters. To do this, we will use the "H1" tag. So in the body add a "H1" tag, your name and closing "H1" tag.

Next thing is we want to insert a line break as we want the four elements on separate lines. To do this, we will add a "BR" tag. Once we do that, we will then insert our image. To insert an image we will type "img src="image.png". Note that unlike most tags, for images there is no closing tag. At the end, add an a break line.

Next thing we are going to use the "H2" tags for your bio and short description. We are using the H2 tags as we do not want as much emphasis compared to your name and we are taking it to the next level down. Add a break at the end.

Finally, add an image icon of the social media account you are using and wrap a link tag around it. Hopefully, your code will look similar to the code that is shown below.

<!DOCTYPE html>
<html>
<head>
<title> Andrew </title>
<link href="style.css" rel="stylesheet"> </head>
<body>
<h1> Andrew </h1><br>
<img src="Andrew.png"><br>
<h2> Short description about myself is that I am a student doing this as a part of my final project</h2><br>
<a href="http://www.google.com"><img src="Facebook.png"></a>
</body>
</html>

If so, congratulations, you have completed making your first webpage! However, if you look at it in the web browser, it looks plain since it is a bunch of text and images sitting on a page with no real design. The next section will show how to make the page look much more appealing.

CSS To Style Page

Open up your "style.css" page that was created earlier as we will now begin to style the page.

The first thing that is going to be done is change the background color and center everything on the page. To do this, we will use the following code shown below:

body{ background-color: black; text-align: center;}

If you preview your webpage, your text is no longer readable since the font color is same as the background. To solve this issue, we will change the color of the H1 and H2 tags as white. Below is code of how to do so.

H1, H2{ color: white;}

Last thing we are going to do is change the font of your name to Helvetica to make it stand out even more. To do this we will use the "font-family" in CSS. Code is shown below of how to do so.

H1{ Font-Family: Helvetica;}

If you have completed all of that, congratulations! Your code should look like the example below:

body{ background-color: black; text-align: center;}
H1, H2{ color: white;}
H1{ Font-Family: Helvetica;}

You have successfully create a simple and colorful web page about yourself using HTML and CSS.

What's New in HTML5 and CSS3

HTML5 and CSS3 is an evolution on HTML4.1 and CSS2. The goal of HTML5 and CSS3 is to provide users a better web experience. Alongisde JavaScript and JQuery (more information about those are provided to members), the web is now more responsive, visually more appealing and a better user experience to all users. Browser Support

One of the most important things to know before starting to write your website in HTML5 and CSS3 is compatibility. Because HTML5 and CSS3 are both relatively new technologies and programming languages, not all browsers support all the elements, attributes and styles that HTML5 and CSS3 brings. As a result, websites developed might not look as expected. There are a few websites that help developers know what elements and attributes are supported. HTML5 Test is a good website to see what elements and attributes are supported by the browser being viewed in and CSS3 Reference by W3 Schools is a good reference to see what styles all browsers support.
